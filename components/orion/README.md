## Orion Component

This folder is the home for **FIWARE Orion Context Broker** specific configuration
and future deployment assets.

- The running instance is defined in the root `docker-compose.yml` as service `orion`.
- Use this folder for:
  - NGSI data models and example payloads
  - Orion-specific config snippets or k8s manifests
  - Notes about how this Pylon talks to upstream/downstream systems
